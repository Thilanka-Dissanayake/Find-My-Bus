# 🚌 Find My Bus 

A real-time tracking application designed to monitor bus locations and estimate arrival times, bridging hardware data collection with user-facing interfaces.

## 🛠️ Built With
*   **Languages:** ( Java)
*   **Backend & Data:** (Firebase Realtime Database)
*   **APIs:** (Google Maps API)

## ⚙️ Features
*   Live geospatial location tracking of transit vehicles.
*   Dynamic data synchronization for real-time updates.

## 🚀 How to Run 
1. Clone the repository: `git clone https://github.com/Thilanka-Dissanayake/Find-My-Bus.git`
2. Add your own API credentials to the configuration file.
